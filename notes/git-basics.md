# Git Basics

- Create a directory on your computer
    (mkdir)
- Change from your current directory and go inside of your new directory
   (cd directory_name_here)
- Create a new repository on Github (or find an existing one!)
- "Clone" this repository to your machine
    git clone https_link_here


# Staging Changes in GIT
- git add . 
    This will add all the folders/files at the current spot in the directory to the staging area. 
- git commit -m "This is where you can write a custom message!"
    Making these messages clear and concise is good practice. Not too much information, but definitely describe what you are doing!
- git push
    This will then push all the staged commits to Git
